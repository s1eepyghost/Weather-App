# Weather App

## Overview

The Weather App is a web application that provides real-time weather information for a given location. Users can quickly check the current weather conditions, temperature, humidity, wind speed, and other relevant data to plan their activities accordingly.

## Features

- **Current Weather Information**: The app displays the current temperature, weather conditions (e.g., clear sky, rain), humidity, and wind speed.

- **Location-based Forecast**: Users can input their location or allow the app to access their current location for a localized weather forecast.

- **5-Day Forecast**: In addition to the current weather, the app provides a 5-day forecast, allowing users to plan ahead.

- **Responsive Design**: The app is designed to be responsive, ensuring a seamless experience across various devices, including desktops, tablets, and mobile phones.

## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/s1eepyghost/Weather-App.git
   cd weather-app
   ```

2. **Open `index.html`:**
   Open the `index.html` file in your preferred web browser.

3. **Allow Location Access (Optional):**
   If prompted, allow the app to access your location for a more accurate weather forecast.

4. **Input Location (Optional):**
   Users can manually input a location in the provided search bar if they prefer to check the weather for a specific place.

5. **View Weather Information:**
   The app will display the current weather information, including temperature, conditions, and additional details.
